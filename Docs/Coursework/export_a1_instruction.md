# Инструкция по подготовке графической части на формате А1

Выбранный сервис: diagrams.net (draw.io), https://app.diagrams.net/.

Графическая часть подготовлена в виде Mermaid-кода:

- `er_diagram.mmd` - инфологическая модель базы данных;
- `program_flowchart.mmd` - блок-схема алгоритма работы программы.

Инфологическая модель выполнена для переноса на формат А1. Блок-схема алгоритма подготовлена с использованием типовых обозначений ГОСТ 19.701-90: терминатор начала и конца, блок процесса, блок ввода-вывода, блок условия и блок данных.

## Экспорт ER-диаграммы в формат А1

1. Открыть https://app.diagrams.net/.
2. Выбрать место хранения файла, например `Device`.
3. Нажать `Create New Diagram`.
4. Выбрать пустую диаграмму.
5. В меню открыть `Arrange -> Insert -> Advanced -> Mermaid`.
6. Открыть файл `Docs/Coursework/er_diagram.mmd`, скопировать весь текст и вставить его в окно Mermaid.
7. Нажать `Insert`.
8. Открыть `File -> Page Setup`.
9. В разделе `Paper Size` выбрать `A1`, ориентацию установить `Landscape`.
10. Разместить ER-диаграмму по центру листа и при необходимости увеличить масштаб.
11. Выполнить экспорт через `File -> Export as -> PDF`, `SVG` или `PNG`.
12. Для печати выбрать масштаб `Fit to page` или `Fit to A1 page`.

## Экспорт блок-схемы алгоритма в формат А1

1. Открыть https://app.diagrams.net/.
2. Создать новую пустую диаграмму.
3. В меню открыть `Arrange -> Insert -> Advanced -> Mermaid`.
4. Открыть файл `Docs/Coursework/program_flowchart.mmd`, скопировать весь текст и вставить его в окно Mermaid.
5. Нажать `Insert`.
6. Открыть `File -> Page Setup`.
7. Выбрать формат `A1`, ориентацию `Portrait` или `Landscape`. Для данной блок-схемы удобнее использовать `Portrait`, если схема вытягивается по вертикали.
8. Разместить схему на листе, проверить читаемость подписей блоков.
9. Выполнить экспорт через `File -> Export as -> PDF`, `SVG` или `PNG`.
10. При экспорте в PNG установить повышенное разрешение, чтобы схема оставалась читаемой при печати.
